Detailed Description

template<typename T>
class DoubleBuffer< T >

Class that defines a thread-safe double buffering mechanism to handle concurrent reads and writes. Two internal buffers are used, with one writer and multiple readers.

Template Parameters
TType of elements stored in the double buffer.
